usb-boop

Know your USB speed instantly. A tiny macOS menu bar utility that watches for device connections and shows the negotiated link speed.

Download View on GitHub
# Install via Homebrew
brew tap alexcatdad/tap
brew install --cask alexcatdad/tap/usb-boop

Features

Instant detection

IOKit event-driven monitoring detects USB connections the moment they happen. No polling.

Speed at a glance

See negotiated link speed immediately — from USB 1.x at 1.5 Mbps to USB 3.2 Gen 2x2 at 20 Gbps.

Native notifications

macOS notifications with device name, speed, and connection type. Toggle on or off.

Menu bar resident

Lives in the menu bar. No dock icon, no windows to manage. Click to see all connected devices.

Cable & hub testing

Quickly verify if your cable or hub supports the speed you expect. Swap and compare in seconds.

Privacy first

Fully offline, local-only. No analytics, no network calls, no data leaves your Mac.
