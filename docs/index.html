A native macOS menu bar utility that detects USB devices the moment they connect and tells you the negotiated link speed. No digging through System Information.

# install via Homebrew
brew install --cask alexcatdad/tap/usb-boop
Download from Releases | macOS 14+ · Apple Silicon

How it works

usb-boop registers for IOKit attach events and surfaces device name plus negotiated speed as a native macOS notification in under a second. It lives in the menu bar with zero dock presence. Click the icon to see every connected USB device and its current link speed.

Perfect for quickly testing whether a cable, hub, or port actually delivers the speed you expect. Swap and compare in seconds.

Fully offline, local-only. No analytics, no network calls, no data leaves your Mac.
