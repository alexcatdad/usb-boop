# Branding

## App Icon

`usb-boop` currently uses a custom macOS app icon built from a simple in-repo SVG source:

- Warm cream background with a soft amber border
- Dark teal USB glyph as the main motif
- Bold glasses over the center to keep the icon playful and recognizable

Source artwork lives in [app-icon-source.svg](/Users/alex/REPOS/alexcatdad/usb-boop/Design/app-icon-source.svg) and the generated app icon set lives in [AppIcon.appiconset](/Users/alex/REPOS/alexcatdad/usb-boop/Sources/App/Assets.xcassets/AppIcon.appiconset).
